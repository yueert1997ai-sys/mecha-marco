# Changelog

## 4.0.2-device-regression

- 修复基地面板类名误作用于 Overlay 导致的安全区左右不对称
- 重新排布右摇杆周围动作按钮，消除军刀、推进、导弹与摇杆命中区域重叠
- 为右上资源栏预留暂停按钮空间
- 完成 956×440 与 844×390 的真实 Chromium 点击、多点触控、动作按钮和横竖屏测试
- 新增移动布局回归测试并更新 QA 记录

## 4.0.1-iphone17-polish

- 修复 iPhone 17 Pro Max 与大屏 iPhone 横屏时基地界面被底部裁切的问题
- 使用 `dvh` / `svh`、四向安全区、短屏断点和三列机体卡布局适配 Safari 横屏
- 隐藏短屏模式突兀滚动条，压缩标题、对白、状态条和机体卡垂直占用
- 扩大 Canvas 可视范围，降低机体在画面中的相对尺寸
- 增加房间布局识别、机体亮度与轮廓可读性、分层刀光、光束与导弹表现
- 增加敌我受击停顿、镜头震动和冲击环反馈
- 新增大屏 iPhone 横屏浏览器烟雾测试和移动布局静态回归测试
- 更新 Service Worker 缓存版本及新增资源清单

## 4.0.0-core-slice

- 重构为 ES 模块和数据驱动系统
- 新增固定步长循环、事件总线和状态机
- 新增 Hardpoint Rig，统一枪口、军刀、导弹与推进器节点
- 新增 Hades 式可见奖励房间路径
- 新增动作槽位构筑、稀有模块和组合协议
- 新增基地、局外档案、条件对话和失败推进
- 新增三台机体、六类普通敌人、两类精英和三阶段首领
- 新增 iOS 横屏双摇杆触控、独立技能按钮与安全区适配
- 新增系统测试和 GitHub Actions 浏览器烟雾测试
