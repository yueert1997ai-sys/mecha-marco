# Changelog

## 4.1.1-controls-camera-polish

- 重写手机双摇杆输入曲线，缩小死区并降低右摇杆自动射击阈值
- 提高触屏转向、起步、收杆和机体朝向响应，降低受击后的黏滞感
- 允许推进过程中继续主射击，并保留轻微射速代价
- 重做战斗镜头：扩大视野、保持场地居中、加入有限的瞄准方向前视，避免冲刺时画面被甩走
- 玩家出生点向场内收拢，完整机体不再压出战斗区域
- 调整玩家机体三分之四俯视姿态、动态侧倾、持枪和军刀动作
- 增加原创头部传感器、胸口散热结构、肩部机械细节和自动目标提示环
- 手机 WebGL 像素比降至 0.9，普通敌人改用低 draw-call 3D 骨架，玩家、精英和 Boss 保留完整硬表面模型
- 普通敌人补充腰胯、手臂和头部识别结构，避免性能版轮廓退化成简单叉形
- 956×440 DPR 3 双指移动/瞄准、推进射击和截图 QA 通过；SwiftShader 平均帧间隔从约 67ms 降至约 39ms
- 844×390 紧凑横屏继续保持无滚动、无裁切
- Service Worker 缓存升级并增加移动手感、镜头、性能和机体轮廓回归测试

## 4.1.0-full-redesign

- 三台可选机体全部推倒重做，旧版色块拼接机体不再作为正式战斗角色
- 新增 Three.js WebGL 3D 显示层，使用带斜面厚度的装甲网格、金属材质、方向光和边缘线
- 先锋型改为锐利均衡的泛用突击机，具备独立步枪、军刀、胸甲、四肢与推进背包
- 重装型改为宽肩厚胸、重腿、双反应炉背包与大型重武器的堡垒机
- 高机动型改为窄躯干、轻装甲、展开式翼板与多推进节点的截击机
- 敌方普通、近战、狙击、炮击、坦克、精英与首领同步替换为不同结构和比例的 3D 装甲机体
- 新增移动步态、射击后坐、军刀展开、推进焰、受击闪白与战斗姿态联动
- 新增三分之四俯视姿态、深色面板线、接触阴影和独立敌人血条
- 基地机体卡改用三套专属机体预览，移除通用渐变徽章
- WebGL 或 Three.js 不可用时自动启用本地 Canvas 投影装甲兜底
- 完成 956×440、DPR 3 移动端真实浏览器测试，三台机体均实际进入战斗并生成截图
- 更新版本、Service Worker 缓存、文档、WebGL CI 和视觉回归测试
