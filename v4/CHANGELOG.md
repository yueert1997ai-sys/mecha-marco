# Changelog

## 4.1.0-full-redesign

- 三台可选机体全部推倒重做，旧版色块拼接机体不再作为正式战斗角色
- 新增 Three.js WebGL 3D 显示层，使用带斜面厚度的装甲网格、金属材质、方向光和边缘线
- 先锋型改为锐利均衡的泛用突击机，具备独立步枪、军刀、胸甲、四肢与推进背包
- 重装型改为宽肩厚胸、重腿、双反应炉背包与大型重武器的堡垒机
- 高机动型改为窄躯干、轻装甲、展开式翼板与多推进节点的截击机
- 敌方普通、近战、狙击、炮击、坦克、精英与首领同步替换为不同结构和比例的 3D 装甲机体
- 新增移动步态、射击后坐、军刀展开、推进焰、受击闪白与战斗姿态联动
- 新增三分之四俯视姿态、深色面板线、接触阴影和独立敌人血条
- 基地机体卡改用三套专属机体预览，移除通用渐变徽章
- WebGL 或 Three.js 不可用时自动启用本地 Canvas 投影装甲兜底
- 完成 956×440、DPR 3 移动端真实浏览器测试，三台机体均实际进入战斗并生成截图
- 更新版本、Service Worker 缓存、文档、WebGL CI 和视觉回归测试

## 4.0.2-device-regression

- 修复基地面板类名误作用于 Overlay 导致的安全区左右不对称
- 重新排布右摇杆周围动作按钮，消除军刀、推进、导弹与摇杆命中区域重叠
- 为右上资源栏预留暂停按钮空间
- 完成 956×440 与 844×390 的真实 Chromium 点击、多点触控、动作按钮和横竖屏测试
- 新增移动布局回归测试并更新 QA 记录

## 4.0.1-iphone17-polish

- 修复 iPhone 17 Pro Max 与大屏 iPhone 横屏时基地界面被底部裁切的问题
- 使用 `dvh` / `svh`、四向安全区、短屏断点和三列机体卡布局适配 Safari 横屏
- 隐藏短屏模式突兀滚动条，压缩标题、对白、状态条和机体卡垂直占用
- 扩大战斗可视范围，降低机体在画面中的相对尺寸
- 增加房间布局识别、机体亮度与轮廓可读性、分层刀光、光束与导弹表现
- 增加敌我受击停顿、镜头震动和冲击环反馈
- 新增大屏 iPhone 横屏浏览器烟雾测试和移动布局静态回归测试
- 更新 Service Worker 缓存版本及新增资源清单

## 4.0.0-core-slice

- 重构为 ES 模块和数据驱动系统
- 新增固定步长循环、事件总线和状态机
- 新增 Hardpoint Rig，统一枪口、军刀、导弹与推进器节点
- 新增 Hades 式可见奖励房间路径
- 新增动作槽位构筑、稀有模块和组合协议
- 新增基地、局外档案、条件对话和失败推进
- 新增三台机体、六类普通敌人、两类精英和三阶段首领
- 新增 iOS 横屏双摇杆触控、独立技能按钮与安全区适配
- 新增系统测试和 GitHub Actions 浏览器烟雾测试
