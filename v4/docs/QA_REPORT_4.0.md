# 天穹断刃 4.0 Core Slice QA

日期：2026-07-11

- 所有 ES 模块通过 `node --check`
- 17 项 Node 系统测试通过
- 房间图在 5 个固定种子下生成合法
- 同一节点的两个战斗选择不会重复奖励与房间模板
- 奖励卡按门牌动作槽位生成
- 已拥有模块不会重复出现
- 组合协议按标签前置条件解锁
- 枪口位置来自右肩→右前臂→步枪→枪口节点链
- 枪口方向与瞄准方向共线
- 军刀轨迹由左肩→左前臂→刀柄→刀尖节点生成
- 军刀扫掠测试包含 9 个以上独立轨迹点
- 战斗房可以完成波次、进入奖励并返回路线
- 暂停会关闭输入，恢复时清空触控状态
- 失败仍写入永久资源和战役历史
- iOS 安全区、横屏提醒、双摇杆和多按钮结构存在

## 浏览器验证

当前本地容器的 Chromium 进程无法稳定输出 DOM，浏览器检查在本地标记为环境受限，不作为通过项冒充。GitHub Actions 中 `REQUIRE_BROWSER_SMOKE=1`，只有真实加载 ES 模块、渲染基地界面并写入 smoke readiness 标记才会通过。

## 当前范围

这是 4.0 的核心纵切，不宣称 Steam 完整发行版本。后续仍需增加：

- 更完整的区域美术和环境叙事
- 更复杂的障碍物碰撞与导航
- 手柄输入
- 正式音频、美术资产与许可清单
- 30 分钟真机压力测试和 iPhone 多机型专项测试
