- `worldToScreenTopDown` 与 `screenToWorldTopDown` 使用同一纯俯视映射
- 镜头没有斜切、平台深度透视或三分之四战斗投影

- 正式战斗姿态隐藏腿部

## 11. 真实设备风险

自动 QA 使用 Chromium 与 SwiftShader，适合检查逻辑、布局、资源和回归，不代表实体 iPhone 的最终表现。

主要风险：

- WebKit 对 WebGL、触控事件和页面恢复的行为差异
- 高 DPR 和长时间战斗造成 GPU 发热与降频
- Safari 后台切换后 WebGL Context 丢失
- Home Indicator 区域多指操作误触
- CDN 加载失败导致进入 Canvas 兜底
- 程序化模型和特效在真机上的 draw call / 内存压力
