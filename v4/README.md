# Mecha Marco / 天穹断刃 4.1

面向 iPhone Safari 与桌面浏览器的机甲动作肉鸽核心纵切。4.1 保留现有战斗、房间、构筑和局外成长系统，彻底替换旧的色块拼接机体渲染。

## 当前纵切

- 三台差异化机体
- 主射击、军刀、推进、重火力、超限五个动作槽位
- 30 个基础/稀有模块与 6 个组合协议
- 10 深度房间图，奖励在进入房间前可见
- 普通战、精英战、商店、维修、事件与三阶段区域首领
- 敌人预警、受击硬直、方向性击退、受击停顿和阶段状态机
- 基地、战役报告、永久资源与条件对话
- 键鼠和 iPhone 横屏双摇杆/多按钮操作
- iPhone 17 Pro Max 级大屏横屏安全区、动态视口与短屏布局适配
- 无第三方美术、音频或运行库依赖

## 4.1 全机体视觉重制

- 旧版 Canvas 线段、多边形和纯色块机体不再作为正式角色渲染
- 新增投影硬表面装甲网格：每块装甲具备顶面、侧面、背面明暗关系和面板线
- 先锋型重做为锐利、均衡的泛用突击机，保留右手步枪和左侧军刀挂点
- 重装型重做为宽肩、厚胸、重腿、双背包推进器和大型武器的堡垒机
- 高机动型重做为窄躯干、轻装甲、展开式翼板和高亮推进节点的截击机
- 敌方普通、近战、狙击、炮击、坦克、精英与首领同步改用不同轮廓的装甲机体
- 三台机体采用白/灰/深色骨架为主，识别色仅用于胸部、传感器、推进器和武器细节
- 新模型继续读取现有 Hardpoint Rig，枪口、军刀刀柄/刀尖与实际攻击判定保持一致
- 待机、移动步态、射击后坐、军刀挥动、推进焰和受击闪白继续由战斗状态驱动
- 基地选择卡改为三台新机体的专属线稿预览，不再显示通用渐变徽章

## 4.0.1–4.0.2 移动端修正

- 基地界面在短高度横屏下保持三台机体完整可见，底部按钮避开 Home Indicator
- Canvas 视野扩大，玩家与敌人相对画面尺寸降低
- 房间布局增加专属地面标识和环境光差异
- 光束、导弹、军刀加入更清晰的核心、拖尾和分层刀光
- 敌我命中加入短促 hit-stop、镜头震动和冲击环
- 基地面板安全区、右摇杆动作按钮与暂停键重叠问题已通过真实 Chromium 交互测试修复

## 架构

```text
src/
  core/      固定步长循环、数学、事件总线、状态机
  combat/    Hardpoint Rig、技能缓冲、命中与反馈打磨
  actors/    玩家机体与敌人状态机
  run/       房间图与奖励解析
  meta/      存档和条件叙事
  render/    Canvas 2.5D、投影装甲网格、机体设计与表现增强
  input/     键鼠与 iOS 多指触控
  ui/        基地、机体预览、路线、构筑、商店、结果界面
  data/      机体、模块、敌人、房间、对白数据
```

## 验证

```bash
npm run verify
```

验证包括语法检查、系统测试、移动布局回归、机体视觉设计回归、战斗房流程、暂停恢复、存档、房间图、奖励槽位、枪口硬点和军刀扫掠测试。GitHub Actions 还会运行桌面 Chromium 与 956×440 大屏 iPhone 横屏浏览器检查。

浏览器测试可以确认模块加载、Canvas 渲染和交互事件链路，仍不等同于实体 iPhone Safari 的发热、功耗、触觉与长时间稳定性验证。
