# Mecha Marco / 天穹断刃 4.0

面向 iPhone Safari 与桌面浏览器的机甲动作肉鸽核心纵切。4.0 以《Hades》的闭环结构为参考，重新实现战斗动作、房间路径、构筑、局外成长和条件对话。

## 当前纵切

- 三台差异化机体
- 主射击、军刀、推进、重火力、超限五个动作槽位
- 30 个基础/稀有模块与 6 个组合协议
- 10 深度房间图，奖励在进入房间前可见
- 普通战、精英战、商店、维修、事件与三阶段区域首领
- 敌人预警、受击硬直、方向性击退、受击停顿和阶段状态机
- 基地、战役报告、永久资源与条件对话
- 键鼠和 iPhone 横屏双摇杆/多按钮操作
- iPhone 17 Pro Max 级大屏横屏安全区、动态视口与短屏布局适配
- 无第三方美术、音频或运行库依赖

## 4.0.1 打磨项

- 基地界面在短高度横屏下保持三台机体完整可见，底部按钮避开 Home Indicator
- Canvas 视野扩大，玩家与敌人相对画面尺寸降低
- 机体亮度、饱和度与轮廓对比增强，受击时有明显闪白
- 房间布局增加专属地面标识和环境光差异
- 光束、导弹、军刀加入更清晰的核心、拖尾和分层刀光
- 敌我命中加入短促 hit-stop、镜头震动和冲击环

## 架构

```text
src/
  core/      固定步长循环、数学、事件总线、状态机
  combat/    Hardpoint Rig、技能缓冲、命中与反馈打磨
  actors/    玩家机体与敌人状态机
  run/       房间图与奖励解析
  meta/      存档和条件叙事
  render/    Canvas 2D 程序化 2.5D 渲染与表现增强
  input/     键鼠与 iOS 多指触控
  ui/        基地、路线、构筑、商店、结果界面
  data/      机体、模块、敌人、房间、对白数据
```

## 验证

```bash
npm run verify
```

验证包括语法检查、原有 17 项系统测试、新增 4 项移动布局与打磨模块测试、战斗房流程、暂停恢复、存档、房间图、奖励槽位、枪口硬点和军刀扫掠回归测试。GitHub Actions 还会运行桌面 Chromium 与 956×440 大屏 iPhone 横屏 UA 烟雾测试。

浏览器 UA 烟雾测试只能确认页面加载、模块执行和适配资源接入，不等同于 iPhone 真机触控、发热与长时间稳定性验证。
