# Mecha Marco / 天穹断刃 4.1

面向 iPhone Safari 与桌面浏览器的机甲动作肉鸽核心纵切。4.1 保留现有战斗、房间、构筑和局外成长系统，彻底替换旧的色块拼接机体显示。

## 当前纵切

- 三台差异化机体
- 主射击、军刀、推进、重火力、超限五个动作槽位
- 30 个基础/稀有模块与 6 个组合协议
- 10 深度房间图，奖励在进入房间前可见
- 普通战、精英战、商店、维修、事件与三阶段区域首领
- 敌人预警、受击硬直、方向性击退、受击停顿和阶段状态机
- 基地、战役报告、永久资源与条件对话
- 键鼠和 iPhone 横屏双摇杆/多按钮操作
- iPhone 17 Pro Max 级大屏横屏安全区、动态视口与短屏布局适配
- Three.js 0.149.0 WebGL 显示层；网络或 WebGL 不可用时自动回退到本地 Canvas 版本

## 4.1 全机体视觉重制

- 正式战斗画面改为真实 WebGL 3D 机体，不再使用旧版线段、圆形和纯色多边形作为主角色
- 使用带斜面厚度的 `ExtrudeGeometry` 装甲、金属材质、方向光、边缘线和接触阴影
- 先锋型重做为锐利、均衡的泛用突击机，具备独立胸甲、肩甲、腰甲、四肢、背包、步枪和军刀
- 重装型重做为宽肩、厚胸、重腿、双反应炉背包和大型重武器的堡垒机
- 高机动型重做为窄躯干、轻装甲、展开式翼板和多推进节点的截击机
- 敌方普通、近战、狙击、炮击、坦克、精英与首领同步使用不同结构和比例的 3D 装甲机体
- 三台机体采用白/灰/深色骨架为主，识别色只用于胸部、传感器、推进器和武器细节
- 新模型继续读取战斗状态：移动步态、射击后坐、军刀展开、推进焰和受击闪白实时联动
- 3D 层独立恢复敌人血条，并为机体增加接触阴影和三分之四俯视姿态
- 基地选择卡改为三台新机体的专属预览，不再显示通用渐变徽章
- Canvas 投影装甲版本仅作为 Three.js 加载失败或 WebGL 不可用时的本地兜底

## 4.0.1–4.0.2 移动端修正

- 基地界面在短高度横屏下保持三台机体完整可见，底部按钮避开 Home Indicator
- 战斗视野扩大，玩家与敌人相对画面尺寸降低
- 房间布局增加专属地面标识和环境光差异
- 光束、导弹、军刀加入更清晰的核心、拖尾和分层刀光
- 敌我命中加入短促 hit-stop、镜头震动和冲击环
- 基地面板安全区、右摇杆动作按钮与暂停键重叠问题已通过真实 Chromium 交互测试修复

## 架构

```text
src/
  core/      固定步长循环、数学、事件总线、状态机
  combat/    Hardpoint Rig、技能缓冲、命中与反馈打磨
  actors/    玩家机体与敌人状态机
  run/       房间图与奖励解析
  meta/      存档和条件叙事
  render/    Canvas 场景、WebGL 3D 机体、材质灯光、兜底机体与表现增强
  input/     键鼠与 iOS 多指触控
  ui/        基地、机体预览、路线、构筑、商店、结果界面
  data/      机体、模块、敌人、房间、对白数据
```

## 验证

```bash
npm run verify
```

验证包括语法检查、系统测试、移动布局回归、机体设计回归、WebGL 启动、Canvas 兜底、战斗房流程、暂停恢复、存档、房间图、奖励槽位、枪口硬点和军刀扫掠测试。

真实浏览器 QA 使用 956×440、DPR 3、移动端触控环境，要求 Three.js 和 WebGL 实际启动，分别选择先锋、重装和高机动机体进入战斗并保存截图，同时检查 844×390 紧凑横屏布局。该测试仍不等同于实体 iPhone Safari 的发热、功耗、触觉与长时间稳定性验证。
