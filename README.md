# Mecha Marco / 天穹断刃

面向 iPhone Safari 与桌面浏览器的纯网页俯视角机甲动作肉鸽。

## 在线入口

- 正式入口：`https://yueert1997ai-sys.github.io/mecha-marco/`
- 当前 v4：`https://yueert1997ai-sys.github.io/mecha-marco/v4/`
- v3.2 旧版回退：`https://yueert1997ai-sys.github.io/mecha-marco/v3/`

仓库根目录会自动进入当前 v4；旧版运行资源仍保留，便于兼容问题回退。
