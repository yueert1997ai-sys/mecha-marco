# Mecha Marco / 天穹断刃

纯网页 2.5D 超级系机甲动作肉鸽实验项目。

## 当前内容

- `v3.2 iOS 触控版`：用于 iPhone Safari 横屏试玩
- `docs/HADES_ARCHITECTURE_4.0.md`：基于《Hades》核心循环整理的 4.0 产品与代码架构基线

## 当前阶段

仓库先保存现有可玩版本与设计基线。下一阶段将按 4.0 架构重构战斗、房间路径、构筑系统、敌人状态机与局外成长，不再继续向单文件原型无序叠加功能。

## GitHub Pages

仓库根目录包含 `index.html`。开启 GitHub Pages 后即可在 Safari 中直接访问。

> 注意：当前仓库是私有仓库。GitHub Pages 是否可用取决于账号套餐；最省事的方案是将仓库改为 Public。
